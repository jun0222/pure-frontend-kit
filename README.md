## pure-frontend-kit

- ピュアな HTML/CSS/JavaScript のみで構成されたフロントエンド開発ライブラリ。
- Github: https://www.npmjs.com/package/pure-frontend-kit
- npm: https://github.com/jun0222/pure-frontend-kit

## インストール

```bash
npm install pure-frontend-kit
```
